# Social Network Project - Backend

A backend API for an Instagram-like social media application, built using Node.js, Express, and MongoDB.
Supports authentication, posts, follows, chats (realtime via Socket.io), and image uploads using Cloudinary

# Features

- Authentication:
Register / Login with JWT
Secure password hashing (bcrypt)


- Posts:
Create / delete post
Get feed (following + self)
Like / Unlike posts

- Chat (Realtime):
Send / receive messages instantly via Socket.io
Support text and image messages

- User System:
Follow / unfollow users
View other profiles
Search users by username

- Image Upload:
Upload posts and chat images using Cloudinary
